# OpenAPI Definition for ECHE List API

This tool uses the official Redocly CLI Docker image.

## Quick start

Inside the `redoc` directory, where this file is located, the following commands are available:

- `make build-docs` will generate a Redoc preview which can be viewed in a local HTML file;
- `make bundle` will generate the complete specification in YAML and copy it to the Flask assets;
- `make lint` will run the Redoc linter on the specification.
