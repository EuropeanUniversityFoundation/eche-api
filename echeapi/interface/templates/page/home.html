Welcome to the ECHE List API

Higher education institutions holding an ECHE - 2021-2027

This API exposes all Institutions that currently hold the Erasmus Charter for Higher Education (opens in a new tab) as published by the European Commission.

All the data fields contained in the original list are made available to client applications. Additionally, some processed values are also provided to respond to more specific use cases.
